Strong Tag

The < b > and <strong > tags in HTML both make text Bold but they have different meanings and use cases

  1. < b > (bold)
    • Used purely for visual styling.
    • Does not add importance or meaning to the text.
    • Example: < b > Warning < /b > This action cannot be undone.
    • This just makes Warning bold but does not indicate that it's important.
  2. < strong > (strong)
    • Used to indicate that the text is important or has strong emphasis
    • Screen readers announce it differently , helping visually impaired users
    • Example: <strong > Warning </strong >This action cannot be undone.
    • Here, Warning is not just bold but also semantically important .
